# Landing Page Production Plan

## Project Goal

Create a product-catalog-style landing page for **FIM-92 Stinger** as an educational and historical product overview page.

This project is for a web design assignment. It must not include operational instructions, firing procedures, tactical guidance, acquisition methods, modification methods, or any content that enables real-world weapon use.

---

## Required Folder Structure

- landing-page/
  - index.html : Korean page
  - en.html : English page
  - style.css
  - script.js
  - images/
    - stinger-real.jpg
    - defense-concept-1.jpg
    - defense-concept-2.jpg
    - defense-concept-3.jpg

---

## Fixed Requirements

1. The page must be a product introduction landing page.
2. The design concept must be a premium product catalog.
3. The project must include both Korean and English pages.
4. Korean page: `index.html`
5. English page: `en.html`
6. Both pages must share the same visual design.
7. The website must be responsive for desktop, tablet, and mobile.

## Safety / Content Rules

### Prohibited Content

- firing procedures
- aiming or targeting instructions
- tactical deployment guidance
- performance optimization for real-world use
- acquisition or purchase guidance
- modification or bypass methods
- combat advice

### Allowed Content

- general product overview
- historical background
- non-operational specifications
- educational description
- catalog-style visual presentation
